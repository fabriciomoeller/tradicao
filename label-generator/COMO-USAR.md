# 🎯 COMO USAR - Gerador de Etiquetas Tradição

## 🚀 Início Rápido (2 minutos)

### 1. Abrir o Aplicativo
- Abra o arquivo `index.html` no seu navegador
- A interface carregará com configurações padrão prontas

### 2. Teste Imediato
- Clique em **"Gerar Etiquetas"** (já tem valores configurados)
- Veja a visualização na tela
- Clique em **"Imprimir"** para testar

### 3. Primeiro Resultado
✅ 10 etiquetas geradas automaticamente:
- 5 tickets de R$ 5,00
- 3 tickets de R$ 10,00  
- 2 tickets de R$ 20,00

---

## ⚙️ Configuração Personalizada

### Dimensões da Etiqueta
```
Largura: 7cm (padrão) - ajuste conforme necessário
Altura: 3.5cm (padrão) - ideal para tickets
```

### Margens da Página
```
Superior/Inferior: 1cm - espaço das bordas
Esquerda/Direita: 1cm - alinhamento da impressora
```

### Aparência do Texto
```
Tamanho: 14px (padrão) - legível e elegante
Fonte: Arial - compatível com todas as impressoras
Cor: Preto - ou personalize conforme o evento
```

---

## 💰 Configurando Valores dos Tickets

### Adicionar Novos Valores
1. **Digite o valor** em reais (ex: 15.50)
2. **Defina a quantidade** de etiquetas
3. **Clique "Adicionar Valor"** para mais tipos
4. **Use "Remover"** para excluir valores desnecessários

### Exemplo Prático - Festa Junina
```
R$ 2,00 → 100 tickets (comidas baratas)
R$ 5,00 → 50 tickets (pratos principais)  
R$ 10,00 → 30 tickets (bebidas/doces especiais)
R$ 20,00 → 10 tickets (prêmios/rifas)
```

---

## 🖼️ Adicionando Imagem de Fundo

### Como Adicionar
1. **Clique em "Imagem de Fundo"**
2. **Selecione sua imagem** (PNG, JPG)
3. **Ajuste a opacidade** (recomendado: 0.3)
4. **Regenere as etiquetas**

### Dicas para Imagens
- **Logo da organização**: Opacidade 0.2-0.3
- **Imagens coloridas**: Opacidade 0.1-0.2  
- **Marcas d'água**: Opacidade 0.4-0.5
- **Tamanho ideal**: Máximo 1MB para performance

## 🖨️ Impressão Profissional

### Configuração da Impressora
```
Papel: A4 (21cm × 29.7cm)
Margens: Mínimas ou Nenhuma
Qualidade: Normal/Alta
Orientação: Retrato
```

### Processo de Impressão
1. **Revise a visualização** final
2. **Clique "Imprimir"** 
3. **Configure a impressora** conforme acima
4. **Imprima uma página teste** primeiro
5. **Ajuste se necessário** e imprima o restante

### Papel Recomendado
- **Tickets comuns**: Papel A4 75-90g/m²
- **Eventos importantes**: Papel A4 120-150g/m²
- **Etiquetas adesivas**: Papel A4 adesivo
- **Cartões**: Papel cartão 180-250g/m²

## 🔧 Solução de Problemas Comuns

### ❌ "Etiquetas não aparecem"
**Solução:**
- ✅ Clique em "Gerar Etiquetas"
- ✅ Verifique se há valores configurados
- ✅ Confirme dimensões válidas

### ❌ "Impressão cortada"
**Solução:**
- ✅ Aumente as margens (1.5cm ou mais)
- ✅ Configure impressora para margens mínimas
- ✅ Teste com página em branco primeiro

### ❌ "Imagem não aparece"
**Solução:**
- ✅ Use formatos JPG ou PNG
- ✅ Verifique se opacidade não está em 0
- ✅ Teste com imagem menor (< 1MB)

### ❌ "Muitas/poucas etiquetas por página"
**Solução:**
- ✅ Ajuste dimensões das etiquetas
- ✅ Modifique margens da página
- ✅ Veja preview antes de imprimir

## 💡 Dicas de Profissional

### 🎨 Design Efetivo
- **Contraste adequado**: Texto escuro em fundo claro
- **Legibilidade**: Fontes mínimo 12px para impressão
- **Hierarquia visual**: Valor em destaque, info adicional menor
- **Espaços em branco**: Não sobrecarregue as etiquetas

### 📊 Planejamento de Quantidade
- **Calcule 10-20% extra** para reposição
- **Distribua valores conforme demanda** esperada
- **Considere troco** ao definir valores
- **Mantenha valores redondos** para facilitar

### 🔄 Reutilização Inteligente
- **Salve templates** para eventos recorrentes
- **Documente configurações** que funcionaram bem
- **Teste diferentes papéis** para cada tipo de evento
- **Mantenha backup** dos arquivos de configuração

### ⚡ Performance
- **Lotes menores** (até 500 etiquetas) são mais rápidos
- **Imagens otimizadas** melhoram a performance
- **Feche outras abas** do navegador durante impressão
- **Use modo incógnito** se houver problemas de memória
